- `/bin,/usr/bin,/usr/local/bin`

##### foregin filesystem mountpoints    
- `/media,/mnt  `  

##### linux file hierarchy concepts  
- /etc						-system config files    
- /tmp						-temporary files    
- /boot						-kernel and bootloader    
- /dev						-device    
- /usr						-programs    
- /lost+found	    
- /var,/srv					-server data    
- /proc						-system information    
- /lib,/usr/lib,/usr/local/lib -libraries
