循环队列需要几个参数来确定？
	2个: fornt, rear

这两个参数在不同场合有不同含义：
	1. 队列初始化
	   fornt和rear对应的值都是零
	2. 队列非空
	   front代表的是队列的第一个元素
	   rear代表的是队列的最后一个有效元素的下一个
	3. 队列空 
	   front和rear对应的值相等，但不一定是零

注意：循环队列首位相连，由首指向尾，下一个位置始终是下标大一的位置(交界处例外)
入队算法：
	1. 将值存入r所代表的位置
	2. r向下一个位置移动。r = (r + 1) % 数组的长度

出队算法：
	1. 将f的值删除 // 这一步可以略去
	2. 将f向下一个位置移动。f = (f + 1) % 数组的长度

如何判断循环队列是否为空:
	如果front与rear对应的值为同一个元素该队列就一定为空

 如何判断循环队列是否已满：
	两种方式：
		1. 多增加一个标识参数，记录元素个数
		2. 少用一个元素。
	一般用第二种方法，定义n(数组长度为n)个元素，只允许使用n-1个
	即：若front和rear对应的值紧挨着，则循环队列已满，否则未满
	if ( (r+1)%n == f)
	    已满;
	else
	    未满;

队列的应用：
	所有和时间有关的操作都与队列有关
